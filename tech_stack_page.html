Our Technology Stack

Core Infrastructure

AdFlash is designed for scalability, speed, and reliability. We use a modern, full-stack architecture with secure cloud deployment.

  • Frontend: Next.js (React), TailwindCSS, shadcn/ui
  • Backend: Node.js, Express, REST API
  • Database: PostgreSQL + Prisma ORM
  • Authentication: Auth.js with Google + Email/Password + Admin role support
  • Storage: Cloudflare R2 for image assets, Supabase Storage fallback
  • Deployment: Vercel (Frontend), Railway or Render (Backend/API)

AI & Generation Engines

  • Text Generation: GPT-4-turbo (OpenAI API)
  • Image Generation: GPT-4o / gpt-image-1 (Vision-based image rendering)
  • Prompt Engine: Modular JSON prompt builder with creative intent, audience, and brand tone parameters
  • Layout Presets: Library of over 500+ ad templates mapped to objectives and industries
  • Performance Logic: Ad Matrix for validating persuasive hooks, CTAs, and layouts

Analytics & Monitoring

  • Post-generation performance matrix to analyze expected impact
  • Integration-ready for Meta Pixel and Google Ads Conversion API
  • Error monitoring via Sentry and uptime monitoring via Cronitor
